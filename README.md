## Project Tech Stack

* Xcode: Version 14.2 (14C18)

* Language: Swift 5.7.2

* Minimum iOS Version: 13.0

* Design Pattern: MVVM

* Dependency Manager: Swift Package Manager

* Dependencies: Alamofire, SnapKit, Kingfisher
